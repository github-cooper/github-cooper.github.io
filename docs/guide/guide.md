# Getting Started with VuePress

---

> [VuePress Tutorial](./tutorial/vuepress-tutorial.md)
>
> 

[VuePress](https://v2.vuepress.vuejs.org/) is a markdown-centered static site generator. You can write your content (documentations, blogs, etc.) in [Markdown](https://en.wikipedia.org/wiki/Markdown), then VuePress will help you to generate a static site to host them.

The purpose of creating VuePress was to support the documentation of Vue.js and its sub-projects, but now it has been helping a large amount of users to build their documentation, blogs, and other static sites.
