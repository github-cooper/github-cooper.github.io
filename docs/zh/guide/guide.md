# VuePress 介绍

[VuePress](https://v2.vuepress.vuejs.org/zh/) 是一个以 [Markdown](https://markdown.com.cn/) 为中心的静态网站生成器。可以使用 [Markdown](https://en.wikipedia.org/wiki/Markdown) 来书写内容（如文档、博客等），然后 VuePress 会帮助你生成一个静态网站来展示它们。

VuePress 诞生的初衷是为了支持 Vue.js 及其子项目的文档需求，但是现在它已经在帮助大量用户构建他们的文档、博客和其他静态网站。
